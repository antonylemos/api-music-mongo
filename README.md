# API REST to Mongo

Application developed for the NoSQL Database discipline in BTI.

## Installation

Use the [yarn](https://yarnpkg.com/) to install.

```bash
git clone https://github.com/lemosantony/api-music-mongo.git
cd api-music-mongo
yarn install
```

Or

```bash
git clone https://github.com/lemosantony/api-music-mongo.git
cd api-music-mongo
npm install
```

## Run

```bash
yarn dev
```

Or

```bash
npm dev
```
